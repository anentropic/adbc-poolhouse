```python
from __future__ import annotations
from typing import Self
from pydantic import field_validator, model_validator
from pydantic_settings import SettingsConfigDict
from adbc_poolhouse._base_config import BaseWarehouseConfig
from adbc_poolhouse._exceptions import ConfigurationError  # noqa: TC001

class DuckDBConfig(BaseWarehouseConfig):
    model_config = SettingsConfigDict(env_prefix="DUCKDB_")
    database: str = ":memory:"
    pool_size: int = 1   # default 1 for in-memory
    read_only: bool = False
    # field_validators for pool_size, max_overflow, timeout, recycle, database
    def _adbc_entrypoint(self) -> str | None:
        return "duckdb_adbc_init"
    @model_validator(mode="after")
    def check_memory_pool_size(self) -> Self: ...
```

From src/adbc_poolhouse/_duckdb_translator.py (translator template):
```python
from __future__ import annotations
from typing import TYPE_CHECKING
if TYPE_CHECKING:
    from adbc_poolhouse._duckdb_config import DuckDBConfig

def translate_duckdb(config: DuckDBConfig) -> dict[str, str]:
    kwargs: dict[str, str] = {"path": config.database}
    if config.read_only:
        kwargs["access_mode"] = "READ_ONLY"
    return kwargs
```

Create src/adbc_poolhouse/_sqlite_config.py mirroring _duckdb_config.py exactly, with these differences:

1. env_prefix="SQLITE_" (not "DUCKDB_")
2. No read_only field — SQLite ADBC driver does not expose a read-only kwarg in db_kwargs
3. _adbc_entrypoint() returns "adbc_driver_sqlite_init" (not "duckdb_adbc_init")
4. Error message in check_memory_pool_size must say "in-memory SQLite is shared across all connections" (NOT "isolated per connection" — that is DuckDB's behaviour, not SQLite's)

6. Include all field_validators (pool_size, max_overflow, timeout, recycle) from DuckDBConfig

CRITICAL docstring note: SQLite's :memory: database is shared across all connections in the pool — the opposite of DuckDB which gives each connection an isolated empty DB. The guard exists because pool_size > 1 with a single shared in-memory DB is almost always unintended (connection state races).

Google-style docstrings required (CLAUDE.md quality gate):
- Class docstring with Args/Examples sections
- Field docstrings as inline strings after each field
- Docstring for _adbc_entrypoint() (can be brief — internal method)
